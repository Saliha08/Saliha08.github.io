Review Cards

What is HTML?

HTML is used to create the structure of web pages.

What is CSS?

CSS is used to style web pages.

What is JavaScript?

JavaScript is used to add interactivity to web pages.

What is a Browser?

A browser allows you to view websites.

What is Responsive Design?

Responsive design makes web pages look good on all devices.

What is Flexbox?

Flexbox is a CSS tool for creating flexible layouts.

What is a DOM?

DOM is the structure of a web page that JavaScript can interact with.

What is a Web Server?

A web server hosts websites and sends data to browsers.
